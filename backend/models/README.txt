Place your GGUF model file here.

Recommended free models (download from Hugging Face):
- mistral-7b-instruct-v0.2.Q4_K_M.gguf   (~4 GB)
- llama-2-7b-chat.Q4_K_M.gguf            (~4 GB)
- phi-3-mini-4k-instruct.Q4_K_M.gguf     (~2 GB)

After downloading, update LLM_MODEL_PATH in your .env file to point to the file.

Example:
  LLM_MODEL_PATH=models/mistral-7b-instruct-v0.2.Q4_K_M.gguf
